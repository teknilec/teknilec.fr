# teknilec.fr

# A propos

Le site web officiel de Teknilec.

# Développement

-   Démarrer le serveur de dev :

```
yarn dev
```

-   Générer un build de production :

```
yarn build
```

-   Tester le build de production en local :

```
yarn start
```
